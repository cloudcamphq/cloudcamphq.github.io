camp init

Initialize a new CloudCamp project. Running this command creates all files necessary for deploying your app on AWS.

Usage

$ camp init

Arguments

-h,
--help

Show CLI help.

-n,
--name=name

The name of your app.

--yes

Accept the default choices.

This is some more documentation.

--debug

Debug mode. This is only useful for working on the cloudcamp source.

Examples

This will create an app in the directory cloudcamp/myapp:

$ camp init --name=myapp
